# Python Mini Projects

A collection of beginner-to-intermediate Python projects built in 
Google Colab, covering core programming concepts through 
interactive games and logic-based challenges.

## Projects

### 🎯 Number Guessing Game
A command-line game where the player tries to guess a randomly 
generated number within a set number of attempts. Covers random 
number generation, loops, and conditional logic.

### ❓ Quiz Game
An interactive multiple-choice quiz application. Covers data 
structures (lists/dictionaries), control flow, and score tracking.

### ✂️ Rock Paper Scissors
A playable Rock-Paper-Scissors game against the computer. Covers 
user input handling, randomisation, and game loop design.

## Tech
Python · Google Colab · Jupyter Notebook

## About
These projects were built as part of my early Python learning 
journey, forming the foundation for later work in data science 
and machine learning.
